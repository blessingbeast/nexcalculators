# Scalable Calculator Website

A scalable, SEO-first calculator website built with Node.js, Express, and EJS.

## Features
- Server-side rendering for SEO
- 20+ Calculators (EMI, BMI, Age, etc.)
- Clean, minimal, mobile-responsive UI
- Modular architecture

## Setup
1. `npm install`
2. `npm start`
3. Visit `http://localhost:3000`

## Structure
- `/calculators`: JSON configuration for all calculators
- `/views`: EJS templates
- `/public`: Static assets
