What is a Percentage Change Calculator?

A percentage change calculator measures the relative difference between an old value and a new value, expressed as a percentage. A positive result indicates growth or increase; a negative result indicates a decrease or decline. It's widely used in finance, economics, science, and everyday comparisons.

Formula

% Change = ((New Value − Old Value) / |Old Value|) × 100

A positive result = increase. A negative result = decrease.

Example Calculation

  • Stock price went from ₹150 to ₹180: % Change = ((180-150)/150) × 100 = +20%
  • Petrol price dropped from ₹106 to ₹96: % Change = ((96-106)/106) × 100 = −9.43%
  • Salary from ₹45,000 to ₹54,000: % Change = +20%

FAQs

Is percentage change the same as percentage difference?

No. Percentage change tracks movement from a defined start point. Percentage difference measures the gap between two numbers without a clear "before" and "after."

Can percentage change exceed 100%?

Yes! If a value doubles, the percentage change is 100%. If it triples, it's 200%. There's no upper limit to percentage increase.
