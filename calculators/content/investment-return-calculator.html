What is an Investment Return Calculator?

An investment return calculator measures the performance of an investment by comparing the initial cost to the final value. It computes both the simple ROI (total percentage gain or loss) and CAGR (Compound Annual Growth Rate) for multi-year investments. CAGR is the most accurate way to evaluate long-term investment performance because it accounts for the time value of money.

Formula

ROI (%) = ((Final Value − Initial Investment) / Initial Investment) × 100

CAGR = ((Final Value / Initial Investment)^(1/Years)) − 1

Example Calculation

You invested ₹2,00,000 in a mutual fund 5 years ago. Today it is worth ₹3,50,000.

  • ROI = ((₹3,50,000 − ₹2,00,000) / ₹2,00,000) × 100 = 75%
  • CAGR = (3,50,000 / 2,00,000)^(1/5) − 1 = 1.75^0.2 − 1 = 11.84% per annum

FAQs

What is a good ROI in India?

It depends on asset class. FDs return 6-8%, REITs around 8-10%, equity mutual funds 12-15% CAGR long-term, and gold about 10% CAGR over decades. Equity has the highest long-term return but with volatility.

What is CAGR and why is it better than simple ROI?

CAGR (Compound Annual Growth Rate) is the annualized return that, if compounded annually, gives you the same final value. It allows fair comparison of investments held for different durations.

Does this calculator account for taxes?

No. Returns from equity (held above 1 year) attract 10% LTCG tax above ₹1L gain. Debt fund and FD returns are taxed at your income slab. Factor this in to find true post-tax returns.
