What is Percentage Decrease?

Percentage decrease measures the relative reduction of a value from its original amount. It represents the difference between the old value and the new value as a percentage of the original value. This metric is commonly used in retail sales (discounts), financial analysis (market corrections), and science (reduction in substance mass).

Formula

Percentage Decrease = ((Old Value - New Value) / Old Value) × 100

Alternatively: (Decrease / Original Value) × 100

Example

Scenario: The price of a laptop dropped from ₹80,000 to ₹68,000 during a festival sale.

  • Original Value = ₹80,000
  • New Value = ₹68,000
  • Decrease = ₹80,000 - ₹68,000 = ₹12,000
  • Percentage Decrease = (₹12,000 / ₹80,000) × 100 = 15%
