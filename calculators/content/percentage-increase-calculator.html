What is Percentage Increase?

Percentage increase is a measure of the relative growth of a value compared to its original amount. It expresses the difference between the old value and the new value as a fraction of the old value, converted into a percentage. It is widely used to track stock market gains, population growth, price hikes, and productivity improvements.

Formula

Percentage Increase = ((New Value - Old Value) / Old Value) × 100

Alternatively: (Increase / Original Value) × 100

Example

Scenario: A company's revenue grew from ₹5,00,000 last year to ₹6,50,000 this year.

  • Original Value = ₹5,00,000
  • New Value = ₹6,50,000
  • Increase = ₹6,50,000 - ₹5,00,000 = ₹1,50,000
  • Percentage Increase = (₹1,50,000 / ₹5,00,000) × 100 = 30%
