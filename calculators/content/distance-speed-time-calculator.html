What is the Distance Speed Time Relationship?

The Distance-Speed-Time (DST) relationship is one of the most fundamental equations in physics and everyday life. Cover more distance in less time = higher speed. Drive the same speed longer = more distance covered. These three variables are interdependent — knowing any two reveals the third. This calculator handles all three cases.

Formula

Distance = Speed × Time

Speed = Distance / Time

Time = Distance / Speed

Example Calculation

  • Speed 80 km/h for 3.5 hours → Distance = 80 × 3.5 = 280 km
  • 300 km in 4 hours → Speed = 300/4 = 75 km/h
  • 450 km at 90 km/h → Time = 450/90 = 5 hours

FAQs

What units should I use?

Keep units consistent. If speed is in km/h, time must be in hours, and distance will be in km. If speed is in m/s, time in seconds gives distance in metres.

Does this work for running pace calculations?

Yes. Convert pace (min/km) to speed first: Speed (km/h) = 60 / Pace (min/km). Then use the speed to find time for any distance.
